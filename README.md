# AI Avatar Generator Pro

A simple website template to create custom avatars using AI. Easy to set up and sell!

## Features
- Make avatars in different styles (anime, cartoon, etc.).
- Change size and shape (square, tall, wide).
- Save and download images.
- Works on phone and computer.
- Demo mode: Test without paying for AI.

## How to Use
1. Open `index.html` in a browser.
2. Type what you want (e.g., "blue robot").
3. Pick style, size, shape.
4. Click "Generate" and download!

## Setup for Real AI
- Get a key from OpenAI (free trial).
- Put it in `script.js` where it says 'YOUR_OPENAI_API_KEY_HERE'.
- Now it makes real AI pictures!
